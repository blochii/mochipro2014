『変数』


値を記憶するには変数が必要。変数は自由に名前を付けられる。

変数を使用するには変数宣言を行わなければならない。



データ型

int　　　整数型　　　ex)1, 5
double　浮動小数点型　ex)2.2, 3.0
char　　文字型　　　　ex)a, A　※1文字しか入らない　　etc…

変数宣言

変数を使用するために行うこと、宣言を行うことで変数を使用することができる。
変数に記憶する値によって型を適切に選択する必要がある。

【形式】データ型 変数名;　//データ型と変数名の間には半角スペースを入れる

【使用例①】double weight;　//double型の変数weightを宣言している
【使用例②】int x,y;　//int型の変数xとyを同時に宣言している

----変数に名前を付けるときのルール----

・使える文字はa~z,A~Z,_,0~9

・最初の一文字目は数字から始められない

・C言語の文法で使う言葉は使えない（printf,intなど）

・原則として小文字を使う

・スペースはいれられない、_は使える


代入

変数に値を格納することを代入という。

【形式】変数名 = 値;　

【使用例①】x = 3;　
【使用例②】x = y;　


初期化

変数宣言と代入を同時に行うことを初期化という。

【形式】データ型 変数名 = 値;　

【使用例①】int x = 3;　
【使用例②】char x = 'a';　



『scanf文』


キーボードから入力するために使用する関数


【形式】scanf("変換指定文字",&入力先の変数名);

入力先の変数は既に宣言されているものを使う。


【使用例①】scanf("%d",&x);

【使用例②】scanf("%d %d",&x,&y);　//複数入力するときは、入力した値が順番に変数に入る


scanfの変換指定文字

%c char型の変数に入力
%d int型の変数を入力
%lf double型の変数を入力

double型の変換指定文字はprintfとscanfで少し違う。printfは%f、scanfは%lfを使う。

「i(アイ)」と「l(エル)」の違いに注意！ここでは「l(エル)」。





『数式』



・算術演算子
演算子 使用例 意味
+ x + y xにyを加える
- x - y xからyを引く
* x * y xにyをかける

% x % y xをyで割った余りを求める

・代入演算子
演算子 使用例 意味

『printf文』


コンソールに出力するためにprintf関数を用いる。


【形式】printf("書式文字列",出力する変数名);

出力する変数は既に宣言されているものを使う。

書式文字列内に\nを書くと改行ができる。


【使用例①】printf("Hello World!");　//変数を出力しないから文字だけでいい

【使用例②】printf("xは%dです",x); //変数を出力する場合、
　　　　　　　　　　　　　　　　書式文字列の出力したい場所に変換指定文字を書く

【使用例③】printf("xは%d、yは%dです",x,y);　//変数は順番に変換指定文字の場所に入る


printfの変換指定文字

%c char型の値を出力
%d int型の値を出力
%f double型の値を出力
