『変数』


値を記憶するには変数が必要。変数は自由に名前を付けられる。

変数を使用するには変数宣言を行わなければならない。



データ型

int　　　整数型　　　ex)1, 5
double　浮動小数点型　ex)2.2, 3.0
char　　文字型　　　　ex)a, A　※1文字しか入らない　　etc…

変数宣言

変数を使用するために行うこと、宣言を行うことで変数を使用することができる。
変数に記憶する値によって型を適切に選択する必要がある。

【形式】データ型 変数名;　//データ型と変数名の間には半角スペースを入れる

【使用例①】double weight;　//double型の変数weightを宣言している
【使用例②】int x,y;　//int型の変数xとyを同時に宣言している

----変数に名前を付けるときのルール----

・使える文字はa~z,A~Z,_,0~9

・最初の一文字目は数字から始められない

・C言語の文法で使う言葉は使えない（printf,intなど）

・原則として小文字を使う

・スペースはいれられない、_は使える


代入

変数に値を格納することを代入という。

【形式】変数名 = 値;　

【使用例①】x = 3;　
【使用例②】x = y;　


初期化

変数宣言と代入を同時に行うことを初期化という。

【形式】データ型 変数名 = 値;　

【使用例①】int x = 3;　
【使用例②】char x = 'a';
